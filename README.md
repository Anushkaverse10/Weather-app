# 🌦️ Vibe Check Weather App

A modern, aesthetic weather app built using **HTML, CSS & JavaScript** and designed with a GenZ UI ✨

## 🚀 Live Demo

👉 https://anushkaverse10.github.io/Weather-app/

---

## ✨ Features

- 🌍 Search weather by city
- 📍 Get weather using current location
- 🌡️ Real-time temperature, humidity & conditions
- 🎨 Glassmorphism + gradient UI
- ⚡ Fast & lightweight (no frameworks)

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Glass UI + gradients)
- JavaScript (Fetch API)
